# Combo Plus

Landing page estática inspirada na proposta visual do Combo+ da Alura, desenvolvida com HTML e CSS. O projeto apresenta uma página promocional com banner principal, seções de benefícios, dispositivos compatíveis e rodapé institucional.

## Tecnologias

- HTML5
- CSS3
- Google Fonts (`Inter`)

## Estrutura do projeto

```text
combo_plus/
|- assets/
|- index.html
|- styles.css
|- README.md
```

## Funcionalidades da página

- Seção principal com chamada de assinatura
- Blocos informativos sobre uso em diferentes dispositivos
- Área de destaque para benefícios do combo
- Lista de dispositivos compatíveis
- Rodapé com links informativos

## Como executar

Como este é um projeto estático, basta abrir o arquivo `index.html` no navegador.

No terminal, você pode navegar até a pasta do projeto e abrir o arquivo manualmente no navegador de sua preferência.

## Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- Estruturação semântica com HTML
- Estilização com CSS
- Organização visual de uma landing page
- Uso de imagens e componentes visuais reutilizáveis

## Observações

- Os links da página atualmente apontam para uma página externa de perfil/projeto.
- As imagens utilizadas estão armazenadas na pasta `assets/`.
- Há sinais de problema de codificação de caracteres no conteúdo HTML, o que pode ser ajustado depois se você quiser.
